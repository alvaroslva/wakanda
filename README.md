# Wakanda
Desenvolvimento de um site interativo em homenagem ao terceirão de 2024 da Escola SESI 390.

O projeto reúne fotos dos alunos e foi pensado como uma forma de registrar e celebrar o encerramento do ciclo escolar, incorporando elementos personalizados e interativos.

Entre as funcionalidades desenvolvidas:

- Galeria com fotos dos alunos da turma
- Interações especiais ao clicar em determinados elementos do site
- Easter egg dedicado ao professor homenageado pela turma
- Questionário interativo com surpresa final, incluindo uma mensagem construída coletivamente pela sala

O projeto teve como foco a criação de uma experiência envolvente e personalizada, unindo desenvolvimento web e criatividade.
